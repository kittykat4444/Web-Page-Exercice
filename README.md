# Web-Page-Exercice

This project is a Landing Page built as part of The Odin Project Foundations course. The goal was to recreate a provided design using HTML and CSS, with a strong focus on Flexbox for layout and positioning. Through this project, I practiced structuring web pages, styling components, and creating a visually appealing layout from scratch based on a design mockup.
